# 🎨 VTF/VMT Generator PRO для Garry's Mod NextBots



## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🚀 **Без VTFLib** | Полностью на Python, не требует установки внешних программ |
| 📁 **Пакетная обработка** | Конвертируй сотни текстур за раз |
| 🖼️ **Все форматы** | PNG, JPG, BMP, TGA → VTF (DXT1, DXT5, RGBA8888, RGB888) |
| ⚙️ **Умные VMT** | Автоматическая генерация VMT с настройками для NextBots |
| 👁️ **Предпросмотр** | Смотри изображения прямо в программе |
| 🔍 **Поиск файлов** | Умный поиск по папкам и подпапкам |
| 📋 **Списки файлов** | Сохраняй и загружай списки для повторного использования |
| ✏️ **Пакетное переименование** | Массовое переименование по шаблону |
| 💾 **Резервное копирование** | Автоматическое сохранение оригиналов |
| 🎯 **Точная настройка** | Размер, качество, mipmaps, параметры VMT |
